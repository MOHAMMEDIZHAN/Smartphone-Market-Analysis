# 📱 Smartphone Market Insights Project

## 📌 Project Overview

This project focuses on analyzing a **smartphone market dataset containing 3,100+ records** to uncover pricing trends, brand performance, discount patterns, and product segmentation insights. The analysis demonstrates an **end-to-end data analytics workflow** using **Python for data preprocessing**, **SQL for data extraction and analysis**, and **Tableau for visualization and dashboard development**.

The goal of this project is to simulate a real-world data analytics scenario where raw data is cleaned, transformed, analyzed, and visualized to support data-driven business decisions.

---

## 🎯 Project Objectives

* Clean and preprocess raw smartphone sales data
* Perform data transformation and feature engineering
* Extract business insights using SQL queries
* Build an interactive dashboard for decision-making
* Demonstrate an end-to-end analytics workflow

## 🔄 Project Workflow

### Step 1 — Data Cleaning (Python)

* Handled missing values
* Standardized categorical data
* Converted data types
* Validated pricing and discount calculations
* Removed duplicates

### Step 2 — Feature Engineering

New columns created:

* Price_Category (Budget / Mid-Range / Premium)
* Savings
* Has_Discount
* Price_per_GB
* Rating_Category

### Step 3 — SQL Data Analysis

Developed **25+ SQL queries** ranging from beginner to advanced levels to extract insights.

Key SQL techniques used:

* GROUP BY and Aggregation
* Subqueries
* CASE Statements
* Window Functions (ROW_NUMBER, RANK, AVG)
* Filtering and Sorting
* Business KPI calculations

### Step 4 — Data Visualization (Tableau)

Created an interactive dashboard to visualize market insights.
